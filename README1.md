# 👁️ Face Detection Project
This project is a real-time face detection system
built using Python and OpenCV. It detects human faces 
through a webcam using a pre-trained Haarcascade classifier.

## 🚀 Features
- Real-time face detection using webcam
- Uses Haarcascade (OpenCV)
- Simple and beginner-friendly code
- Can detect multiple faces
 

## 🔍 How It Works
- The webcam captures live video frames  
- Each frame is converted into grayscale for faster processing  
- Haarcascade classifier detects facial features  
- Faces are identified and highlighted with rectangles  
- Output is displayed in real-time  



## 🛠️ Technologies Used

- Python  
- OpenCV  
- Haarcascade XML Classifier  

## ⚙️ Requirements

- Python 3.x  
- OpenCV library (`pip install opencv-python`)  
- Webcam  


## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/payalpr2003-stack/face-detection.git  

2. Go to the project folder:
   cd face-detection  

3. Install dependencies:
   pip install opencv-python  

4. Run the project:
   python emotion.py  

5. Press `Esc` key to exit
